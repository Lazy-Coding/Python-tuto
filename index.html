So, say you want to write a simple program, which takes two numbers, adds them and displays the result; What are the things you'll need to achieve this?
  • First of all, you need to know how a program is structured in that particular language
    For example, A typical C program would look like this,

    #include<stdio.h>     //saves you the trouble of writing low level code

    int main(){    //this is the main entry point for the program
        //do something
        return 0;    //have to return something because of type signature
    }


  • So what about Python? What kind of structure does this language follow? Well, Nothing that grand. What matters the most in python program is Indentation.If you want to know more about python's structure, feel free to google.

    >>> print( "Hello World" )

  • So what if you want to interact with user? like maybe get input from him? So how do you achieve that? In python 3.x , you can take input easily using input() func.

    >>>a=input("enter a number? ")

    enter a number? 15

  • So, let's go a little fancier, Let's write a function!!!! A function in python starts with def keyword, followed by the name of the function, and then parameters enclosed in (). and in the end, a ': ' This marks the start of the function, now from next line, all lines that are indented become the body of the function. Function body ends with the indentation. As always, google if you want to know the specifics.

    def add ( p ,q ) :
        return p+q
    a = input ("Enter a: ")
    b = input ("\nEnter b: ")
    print("\nSum is %d" % add(a,b))

    >>>python3 add.py

    Enter a: 10
    Enter b: 20
    Sum is 30

  • Maybe we forgot something rather important.. Oh yeah, Loops.!. Well this is how you do it,

    for [iterating variable] in [sequence]:
        [do something]

    >>> for i in range ( 0 , 5 ) :
    ...    print(i)
    ...

    0
    1
    2
    3
    4


    Don't take those '...' in account, that is just console going to next line and allowing us to write more lines of code.Note however, that indentation has still been done(4 spaces). That's really important.
